Icon System

Class-based icons for buttons, lists, cards, and SVG sprite usage.

How Developers Use Icons

<button class="btn btn-primary has-icon icon-home">
  Home
</button>

<button class="btn btn-primary has-icon icon-refresh icon-spin">
  Refresh
</button>

<ul class="icon-list">
  <li class="has-icon icon-check">Reusable item</li>
</ul>

SVG Sprite Usage

Use SVG sprite only for complex SVG icons that should not be converted to mask icons.

  <svg aria-hidden="true">
    <use href="#custom-home"></use>
  </svg>
</span>
